# test-module
 * Unit testing | Integration testing | Functional testing (e2e)

## Begin

* Repository containing unit test example, code coverage, create fake module and test configuration in the project

### Pré-requisitos:

*  Node v12.18.2
*  NPM 6.14.8

### Execution

*  npm install
*  npm test
*  npm run coverege
